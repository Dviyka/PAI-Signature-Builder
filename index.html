How To Use

  1. Оберіть тип підпису за допомогою перемикача
  2. Введіть інформацію в поля зліва
  3. Натисніть кнопку «Скопіювати підпис»

Gmail Setup

  1. Відкрийте Gmail → ⚙️ Settings → See all settings
  2. У вкладці General знайдіть блок Signature
  3. Натисніть Create new і вставте скопійований підпис
  4. Оберіть підпис у Signature defaults і натисніть Save Changes
